# Fight X

This game was a try at replicating the gameplay of a SNES game named [Yuu Yuu Hakusho: Tokubetsu Hen][YuuYuuHakushoSNES].

## Info
Year: 2002
Platform: MS-DOS
Language: Free Pascal Compiler
Status: Unfinished
